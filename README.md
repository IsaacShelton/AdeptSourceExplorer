# AdeptSourceExplorer

<img src="https://raw.github.com/IsaacShelton/AdeptSourceExplorer/master/.github/icon.png" width="256" height="256">

Visualize and perform static analysis on your Adept projects

```
npm install
npm run dev # Develop
npm run build # Release
```
